## Development

1. Clone the repository:

    ```shell
    git clone https://github.com/ixn075/Smash.git
    ```

2. To compile the project, run the following command inside the cloned `Smash` directory:

    #### - macOS / Linux
    ```shell
    ./gradlew build
    ```
    #### - Windows
    ```shell
    .\gradlew.bat build
    ```

<hr>

Compiled file is in `build/libs`.
